## Introduction

This application is a simple Todo App built using react js and Hasura as the serverless Backend providing all the GraphQl Api's needed

### Technologies used

- React JS
- React State for state management
- React Hooks
- Hasura
- GraphQl
- Tachyons - for styling
- Apollo Client
